gulp boilerplate

Dependencies
- Node.js

Features:
- SCSS
- PUG
- JavaScript babel
- JSON files to fill content
- "npm run .." command to scaffold the project

How to use it:
- Clone repository
- run: ``chmod -R +x bin/``
- run: ``npm run help``
